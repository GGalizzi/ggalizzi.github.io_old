Últimos Proyectos

Diseño Adaptable

La adaptabilidad en el diseño web es realizar el sitio para que se pueda utilizar de manera optima sin importar el dispositivo en el que se visualice, ya sea un teléfono, tableta, netbook o computadora de escritorio, el diseño se acomoda según que se esta utilizando.

Experiencia de Usuario

El diseño de experiencia de usuario es uno de los elementos mas importantes en el diseño web, se trata de diseñar el funcionamiento y la estructura del sitio para que sea fácil de usar, para que el usuario encuentre lo que busca dentro del sitio de manera rápida y sencilla.

Tipografia

En la web, la gran mayoría del contenido es texto.  El diseño tipográfico optimiza el uso de fuentes, espacio entre letras, renglones y párrafos, y largo de ellos para que leer pueda ser más placentero.

Optimización

Es muy importante que un sitio pueda cargar lo más rápido posible, para que pueda cargar en conexiones mas lentas, especialmente para móviles usando 3G. Esto se logra optimizando solicitudes HTTP, imágenes, HTML, CSS y JavaScript para no desperdiciar tiempo de carga.

Creando aplicaciones con Node-Webkit

Node-Webkit es una aplicación basada en Chromium y Node.js que te permite escribir aplicaciones nativas usando las tecnologías web.

Con Node-Webkit podes escribir aplicaciones de escritorio usando HTML, CSS, JavaScript, y podes usar módulos de Node.js directamente en el DOM. Dado que las tecnologías web cada vez nos otorgan más control sobre nuestras aplicaciones web, esta la posibilidad de hacer aplicaciones de gran complejidad, que en un pasado cercano solo se hubiese pensado que serian cosas solamente posibles con lenguajes usados para hacer aplicaciones de escritorio.
